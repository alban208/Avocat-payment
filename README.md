# Project Payment

## Installation

1. Clone the repository:
    ```sh
    git clone <repository-url>
    cd Project_payment
    ```

2. Install dependencies:
    ```sh
    npm install
    ```

## Running the Application

1. Start the application:
    ```sh
    npm start
    ```

2. Open your browser and navigate to:
    ```
    http://localhost:3000
    ```
